## Поток данных (последовательность)

```
1. Пользователь отправляет команду /update_journal в Telegram
   ↓
2. n8n получает webhook от Telegram Bot
   ↓
3. Узел DOCS_CSV задаёт параметры: проект, папка, имя файла
   ↓
4. DownloadFileFromDocs (Code node):
   4.1. Авторизация в Signal API (clientId + clientSecret + scopes)
   4.2. Поиск проекта по имени
   4.3. Рекурсивный обход папок (folder_name может содержать путь)
   4.4. Получение itemId → versionId → signedUrl
   4.5. Скачивание CSV-файла
   ↓
5. Extract from File: парсинг CSV (разделитель ;, UTF-8 с BOM)
   ↓
6. Parse & Sort CSV (Code node):
   6.1. Очистка ключей от переносов строк
   6.2. Сортировка по дате (формат ДД.ММ.ГГГГ)
   ↓
7. Параллельная запись в две Google-таблицы:
   7.1. Журнал бетонных работ (appendOrUpdate по наименованию конструкции)
   7.2. Журнал ухода за бетоном (appendOrUpdate по наименованию части)
   ↓
8. Параллельные уведомления в Telegram
   ↓
9. Execute Command: запуск Python-скрипта для экспорта в Excel
```

## Структура данных (маппинг CSV → Google Sheets)

## Технологический стек

| Слой | Технология | Назначение |
|------|-----------|------------|
| BIM | Autodesk Revit 2025 | Источник геометрии и параметров |
| Data Platform | Signal (sgnl.pro) | Хранение и API-доступ к данным |
| Integration | n8n (self-hosted) | Оркестрация, ETL, автоматизация |
| Storage | Google Sheets API v4 | Онлайн-таблицы для команды |
| Notifications | Telegram Bot API | Мгновенные уведомления |
| Export | Python 3 + openpyxl | Локальный экспорт в Excel |
| Auth | OAuth2 / JWT | Безопасный доступ к API |

## Особенности реализации

### Рекурсивный обход папок
```javascript
// Поддержка путей: "Папка1/Папка2/Папка3" или "Папка1\\Папка2\\Папка3"
const splitPath = (p) =>
  normalize(p)
    .replaceAll('\\', '/')
    .split('/')
    .map(x => x.trim())
    .filter(Boolean);
```

### Регистронезависимый поиск
```javascript
const sameName = (a, b) => 
  normalize(a).toLowerCase() === normalize(b).toLowerCase();
```

### Два варианта задания пути
1. `folder_name = "6. СМР/Журнал бетонных работ"`, `file_name = "12.csv"`
2. `folder_name = ""`, `file_name = "6. СМР/Журнал бетонных работ/12.csv"`

### AppendOrUpdate (upsert)
- Ключ поиска: наименование конструкции
- Если запись существует — обновляется
- Если нет — добавляется новая строка
- Предотвращает дублирование при повторных запусках
